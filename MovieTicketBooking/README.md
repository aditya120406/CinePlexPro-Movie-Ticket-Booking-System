# 🎬 CinePlex Pro

🚀 Java Movie Ticket Booking System  
💻 Built using Java Swing + OOP  

---

## 🛠️ Tech Stack
- Java
- Swing (GUI)
- OOP Concepts
- File Handling
- Git & GitHub

---

## ✨ Features
- Login System
- City Selection
- Movie Selection
- Show Timings
- Seat Booking System
- Payment Simulation
- Booking Confirmation
